:::info
This page contains information about hardware for running a polkadot/kusama validator node
:::

It is possible to run the polkadot binary on modest hardware. The problem is that when you get in the active set you need plenty of network and cpu resources for validating, especially when para validating.

Polkadot has a benchmark function witch gives you an indication of performance against a “reference” validator. 

The test were done with the command `polkadot benchmark machine` on Ubuntu 20.04 LTS.
